## Overview

This document describes the CNF (configuration) files used in the Auro System.

---

## Key CNF Files

### dcbcnf_A.cnf
- **Purpose:** Discrete Control Block configuration
- **Location:** `D:\Auro\File\dcbcnf_A.cnf`
- **Usage:** Defines DCB parameters for equipment control

### Dcbdef_A.cnf
- **Purpose:** DCB definitions
- **Location:** `D:\Auro\File\Dcbdef_A.cnf`
- **Usage:** Defines DCB structures and parameters

### logdef.cnf
- **Purpose:** Log definitions
- **Location:** `D:\Auro\File\logdef.cnf`
- **Usage:** Configures logging parameters, rotation, retention

### wrm_Auro_a.cnf
- **Purpose:** Warmstart sequence
- **Location:** `D:\Auro\File\wrm_Auro_a.cnf`
- **Usage:** Defines process startup order
